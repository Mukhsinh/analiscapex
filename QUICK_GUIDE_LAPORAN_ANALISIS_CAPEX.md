# Quick Guide: Laporan PDF Analisis Capex

## 🎯 Tentang Laporan Ini

Laporan PDF Analisis Capex adalah laporan ringkas yang fokus pada **data numerik dan tabel perhitungan** untuk analisis keputusan investasi capital expenditure.

### Karakteristik
- ✅ **4 halaman** - Ringkas dan efisien
- ✅ **Data numerik lengkap** - Semua perhitungan detail
- ✅ **Format profesional** - Ukuran huruf optimal dan spasi ringkas
- ✅ **Cepat generate** - Selesai dalam < 3 detik
- ✅ **File kecil** - Ukuran < 100KB

### Perbedaan dengan Laporan Grafik

| Aspek | Laporan Analisis | Laporan Grafik |
|-------|------------------|----------------|
| **Fokus** | Data numerik & tabel | Visualisasi & grafik |
| **Halaman** | 4 halaman | 5-7 halaman |
| **Ukuran** | < 100KB | > 200KB |
| **Konten** | Perhitungan detail | Grafik trend, score, radar |
| **Kecepatan** | < 3 detik | 5-10 detik |
| **Penggunaan** | Analisis detail | Presentasi visual |

## 📥 Cara Download

### Langkah 1: Isi Form Analisis
1. Buka aplikasi Capex Analyzer
2. Isi form **Project Settings**
3. Isi form **Leasing**
4. Isi form **Borrow & Purchase**
5. Isi form **Revenue Sharing**
6. Klik tombol **"Hitung Analisis"**

### Langkah 2: Download PDF
1. Scroll ke bagian **"Hasil Analisis"**
2. Cari section **"Unduh Analisis"**
3. Klik tombol **"Unduh PDF"** (tombol merah)
4. Tunggu proses generate (< 3 detik)
5. PDF akan otomatis terdownload

### Lokasi File
- **Folder**: Downloads (default browser)
- **Nama**: `Laporan-Analisis-Capex-[Equipment]-[Date].pdf`
- **Contoh**: `Laporan-Analisis-Capex-CT-Scan-2026-03-03.pdf`

## 📄 Isi Laporan

### Halaman 1: Cover & Ringkasan Perbandingan

**Konten**:
- Header dengan info project (nama RS, equipment, department)
- Info box (tanggal, periode analisis, discount rate)
- Tabel ringkasan perbandingan 3 alternatif
- Statistik komparatif (min, max, avg, selisih)
- Box rekomendasi dengan alternatif terbaik

**Kegunaan**:
- Executive summary untuk management
- Quick reference untuk keputusan
- Perbandingan cepat antar alternatif

### Halaman 2: Detail Analisis Leasing

**Konten**:
- Tabel pembayaran leasing per tahun
- PV Factor untuk setiap tahun
- PV Expense per tahun
- Total PV Expense

**Kegunaan**:
- Verifikasi perhitungan leasing
- Analisis cash flow tahunan
- Dokumentasi detail untuk audit

### Halaman 3: Detail Analisis Borrow & Purchase

**Konten**:
- Tabel principal payment per tahun
- Interest payment per tahun
- Maintenance expense per tahun
- Total expense per tahun
- PV Factor dan PV Expense
- Trade-in value
- Total Net PV

**Kegunaan**:
- Verifikasi perhitungan purchase
- Analisis komponen biaya
- Evaluasi dampak trade-in

### Halaman 4: Detail Analisis Revenue Sharing

**Konten**:
- Tabel revenue per tahun
- Direct overhead per tahun
- Allocated overhead per tahun
- Operating profit per tahun
- EAT (Earning After Tax)
- PV Factor dan PV Expense
- Total PV Expense

**Kegunaan**:
- Verifikasi perhitungan revenue sharing
- Analisis profitabilitas
- Evaluasi overhead allocation

## 💡 Tips Penggunaan

### Untuk Presentasi Management
1. **Gunakan Halaman 1** untuk executive summary
2. **Highlight rekomendasi** di box hijau
3. **Tunjukkan statistik komparatif** untuk perbandingan cepat
4. **Simpan halaman 2-4** untuk Q&A detail

### Untuk Analisis Detail
1. **Review semua 4 halaman** secara menyeluruh
2. **Verifikasi perhitungan** dengan data input
3. **Bandingkan dengan laporan grafik** untuk insight visual
4. **Dokumentasikan** untuk audit trail

### Untuk Dokumentasi
1. **Simpan PDF** dengan nama yang jelas
2. **Gabungkan dengan laporan grafik** untuk dokumentasi lengkap
3. **Archive** bersama data input (Excel/CSV)
4. **Backup** ke cloud storage

## 🔍 Interpretasi Hasil

### Membaca Tabel Ringkasan

**Kolom "Total PV Expense"**:
- Nilai dalam **Rupiah penuh** (bukan juta)
- Format: Rp 1.234.567.890
- **Semakin rendah semakin baik**

**Kolom "Status"**:
- ✓ Terbaik = Alternatif dengan PV terendah
- Kosong = Alternatif lain

### Membaca Statistik Komparatif

**Total PV Terendah**:
- Nilai minimum dari 3 alternatif
- Ini adalah pilihan terbaik

**Total PV Tertinggi**:
- Nilai maksimum dari 3 alternatif
- Ini adalah pilihan terburuk

**Selisih Min-Max**:
- Perbedaan antara terbaik dan terburuk
- Persentase menunjukkan seberapa signifikan perbedaannya
- Contoh: 15% = perbedaan cukup signifikan

**Rata-rata PV**:
- Nilai tengah untuk referensi
- Berguna untuk evaluasi relatif

### Membaca Rekomendasi

**Box Hijau**:
- Menunjukkan alternatif terbaik
- Berdasarkan total PV terendah
- Penghematan dibanding alternatif terburuk

**Interpretasi**:
- Jika selisih < 5%: Perbedaan tidak terlalu signifikan
- Jika selisih 5-15%: Perbedaan cukup signifikan
- Jika selisih > 15%: Perbedaan sangat signifikan

## ⚠️ Catatan Penting

### Nilai dalam Rupiah Penuh
- Semua nilai ditampilkan dalam **Rupiah penuh**
- **BUKAN** dalam jutaan Rupiah
- Contoh: Rp 1.234.567.890 (bukan 1.234,57 juta)

### Tidak Ada Grafik
- Laporan ini **tidak berisi grafik**
- Untuk visualisasi, gunakan **Laporan Grafik**
- Download dari menu "Laporan dan Grafik"

### Periode Analisis
- Pastikan periode konsisten di semua form
- Biasanya 5-10 tahun
- Sesuaikan dengan umur ekonomis equipment

### Discount Rate
- Gunakan discount rate yang sesuai
- Biasanya 8-12% untuk rumah sakit
- Konsultasikan dengan finance department

## 🆘 Troubleshooting

### PDF Tidak Terdownload
**Solusi**:
1. Cek browser popup blocker
2. Cek permission download di browser
3. Coba browser lain
4. Refresh halaman dan coba lagi

### PDF Kosong atau Error
**Solusi**:
1. Pastikan semua form terisi lengkap
2. Pastikan sudah klik "Hitung Analisis"
3. Cek console browser untuk error
4. Clear cache dan reload

### Nilai Tidak Sesuai
**Solusi**:
1. Verifikasi input data di form
2. Cek perhitungan di layar
3. Bandingkan dengan Excel manual
4. Pastikan discount rate benar

### File Terlalu Besar
**Solusi**:
- Seharusnya < 100KB
- Jika > 200KB, ada masalah
- Report ke developer

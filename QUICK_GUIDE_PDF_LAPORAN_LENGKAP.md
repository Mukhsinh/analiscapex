# QUICK GUIDE: UNDUH PDF LAPORAN LENGKAP

## 🎯 OVERVIEW

Panduan cepat untuk mengunduh laporan analisis CAPEX dalam format PDF yang profesional dan lengkap.

## 📥 CARA MENGUNDUH PDF

### Langkah 1: Pastikan Analisis Sudah Dihitung
1. Isi semua form (Leasing, Purchase, Revenue Sharing)
2. Klik tombol "Hitung Analisis"
3. Tunggu hingga hasil muncul

### Langkah 2: Buka Halaman Laporan Grafik
1. Klik menu "Laporan Grafik" di sidebar
2. Atau navigasi ke `/analisis_capex`
3. Verifikasi grafik ter-render dengan baik

### Langkah 3: Download PDF
1. Klik tombol **"Download PDF"** (biru, di kanan atas)
2. Tunggu loading indicator (⏳ Membuat laporan PDF...)
3. Proses займет 5-15 detik tergantung kompleksitas data
4. File PDF akan otomatis terdownload

### Langkah 4: Buka dan Verifikasi
1. Buka file PDF yang terdownload
2. Periksa kelengkapan laporan
3. Siap untuk dipresentasikan atau dicetak

## 📄 ISI LAPORAN PDF

### Halaman 1: Cover & Executive Summary
- **Header**: Judul laporan dengan branding
- **Info Proyek**: Nama RS, equipment, department
- **Tanggal**: Tanggal pembuatan lengkap
- **Ringkasan Eksekutif**: Analisis singkat hasil
- **Rekomendasi**: Keputusan terbaik (highlight)
- **Tabel Perbandingan**: Ranking 3 alternatif dengan selisih

### Halaman 2: Visualisasi Grafik
- **Grafik Trend**: PV Expense per tahun (line chart)
- **Grafik Score**: Perbandingan score (doughnut chart)

### Halaman 3: Analisis Multi-Kriteria
- **Grafik Radar**: 5 kriteria perbandingan
- **Statistik**: 5 box ringkasan (min, max, selisih, rata-rata, periode)

### Halaman 4: Detail Perhitungan Leasing
- Tabel lengkap per tahun
- Lease Payment, PV Factor, PV Expense
- Total PV Expense

### Halaman 5: Detail Perhitungan Borrow & Purchase
- Tabel lengkap per tahun
- Principal, Interest, Maintenance
- Trade-in value
- Total PV Expense

### Halaman 6: Detail Perhitungan Revenue Sharing
- Summary boxes (Pendapatan, Laba, EAT)
- Breakdown procedures (jika ada)
- Tabel yearly data
- Rincian overhead

### Halaman Akhir: Kesimpulan & Rekomendasi
- **Kesimpulan Akhir**: Ringkasan keputusan
- **Catatan Penting**: 5 poin penting
- **Peringatan**: Khusus untuk Revenue Sharing negative (jika ada)
- **Metodologi**: Penjelasan metode analisis
- **Footer**: Copyright dan timestamp

## 🔧 TROUBLESHOOTING

### Problem: Loading Terlalu Lama (>30 detik)
**Solusi**:
1. Refresh halaman
2. Pastikan koneksi internet stabil
3. Clear browser cache
4. Coba browser lain

### Problem: PDF Tidak Terdownload
**Solusi**:
1. Periksa browser settings (allow downloads)
2. Periksa popup blocker
3. Coba klik kanan > Save As
4. Periksa disk space

### Problem: Grafik Tidak Muncul di PDF
**Solusi**:
1. Tunggu grafik selesai render di halaman
2. Scroll ke semua grafik sebelum download
3. Refresh halaman dan coba lagi

### Problem: Tabel Terpotong
**Solusi**:
1. Ini seharusnya tidak terjadi (auto page break)
2. Jika terjadi, laporkan sebagai bug
3. Screenshot dan kirim ke developer

### Problem: File Size Terlalu Besar
**Solusi**:
1. Normal range: 500KB - 5MB
2. Jika >10MB, mungkin ada issue
3. Coba dengan data yang lebih sederhana
4. Laporkan jika konsisten terjadi
